### 🎯 修改內容

每個文件都包含以下自定義內容：

1. **標題（Title Tag）**
   - 格式：`Goal XX: 中文標題 / English Title`

2. **側邊欄徽章（Badge）**
   - 文字：`GOAL XX`
   - 顏色：每個目標有不同的 Tailwind CSS 顏色類

3. **側邊欄標題**
   - 中文標題（h2）
   - 英文副標題（p）

4. **JavaScript 配置**
   - `storageKey`: 每個文件有獨立的 localStorage 鍵
   - `defaultData.zh`: 中文根節點名稱
   - `defaultData.en`: 英文根節點名稱
   - `defaultData.children`: 4 個預設子節點，每個包含中英文名稱
